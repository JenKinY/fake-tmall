### 目录结构描述
```
├─index.html # 主页html文件
├─pages      # 其他html页面
├─public-resource # 项目使用到的公共资源
    ├─css   # 各个页面的css样式文件
    ├─js    # 各个页面的JavaScript脚本文件
    ├─libs  # 与业务逻辑无关的库
    ├─images# 项目图片文件
```
